# Two-Tier Bookmark Guide

An interactive guide to a two-tier bookmark system:

- **Tier 1:** Bookmark bar for active, high-frequency links.
- **Tier 2:** Raindrop.io for searchable long-term storage.

The guide lets you switch between personal/work contexts, compare tier behaviors, test cleanup cadence, and preview focus folders. Content is data-driven from `src/main.js`.
